# Quantum-Computing-Circuits

This is the repository which contains various quantum circuits, from entanglement to the algorithms such as Deutsch - Jotza algorithm, Shor's algorithm, etc. 
It also contains various QML algorithms. 

Warning: Since qiskit is changing, algorithms may not work on latest versions. Specific versions are required for various algorithms.  

[Index](https://curiousbhakti.github.io/Quantum-Computing-Circuits/) for viewing circuits in the browser using HTML.
